1.4 JavaScript: Interação com o usuário (As portas automáticas, entre outra funcionalidades)

Módulo 1: Introdução ao Desenvolvimento Web

Já temos a estrutura (HTML) e a decoração (CSS) da nossa loja. Agora, com o JavaScript, vamos adicionar interatividade, como portas que abrem e fecham sozinhas, luzes que acendem sozinhas quando se entra em uma sala, música ambiente, provadores interativos, e muito mais. O JavaScript é a linguagem de programação que nos permite criar páginas dinâmicas e interativas, respondendo às ações do usuário e manipulando o conteúdo da página em tempo real.

O que o JavaScript pode fazer?

  • Manipular o conteúdo da página: Alterar textos, imagens, estilos.
  • Responder a eventos: Cliques, movimentos do mouse, pressionamento de teclas.
  • Fazer requisições assíncronas: Buscar informações do servidor sem atualizar a página.
  • Validar formulários: Verificar se os dados inseridos pelo usuário são válidos.
  • Criar animações e efeitos visuais.
  • E muito mais!

Como usar o JavaScript:

Podemos adicionar JavaScript à nossa página de três maneiras, assim como o CSS:

  1. Inline: Diretamente na tag HTML, usando o atributo onclick, onmouseover, etc. (Como colocar um sensor na porta para ela abrir automaticamente)
    <button onclick="alert('Olá!')">Clique aqui</button>
  1. Interno: Dentro da tag <head> ou <body> do HTML, usando a tag <script>. (Como instalar um sistema de som na loja)
     <head>
    <script>
        function mostrarAlerta() {
            alert('Olá!');
        }
    </script>
</head>
<body>
    <button onclick="mostrarAlerta()">Clique aqui</button>
</body>

  1. Externo: Em um arquivo JavaScript separado, linkado ao HTML. (Como contratar um DJ para tocar na loja)
    <head>
    <script src="script.js"></script>
</head>

    E no arquivo script.js:

     function mostrarAlerta() {
    alert('Olá!');
}

Exemplo básico:

<body>
		<p id="meuParagrafo">Este texto vai mudar</p>
    <button onclick="mudarTexto()">Clique aqui</button>
    <script src="script.js"></script>
</body>
 function mudarTexto() {
    document.getElementById("meuParagrafo").innerHTML = "O texto mudou!";
}

Este código define uma função chamada mudarTexto(). Quando essa função é chamada (por exemplo, ao clicar em um botão), ela encontra o elemento HTML com o ID "meuParagrafo" e altera o seu conteúdo para "O texto mudou!".

Para você, jovem iniciante: O JavaScript é uma linguagem poderosa e versátil que permite criar interações incríveis nas suas páginas web. Comece com exemplos simples e, aos poucos, vá explorando as suas funcionalidades. Existem inúmeros recursos online para aprender JavaScript, desde tutoriais básicos até cursos avançados. Pratique bastante, e você verá como o JavaScript pode dar vida aos seus projetos!

Conexão com o projeto EmpreGO!: No nosso projeto, podemos usar o JavaScript para, por exemplo:

  • Criar um sistema de busca dinâmica de vagas.
