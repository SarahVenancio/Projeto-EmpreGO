3.2 Criando o componente de rodapé (footer.html) (Informações de contato e links úteis)

Módulo 3: Desenvolvimento dos Componentes com Bootstrap -

O rodapé é a parte inferior do site, geralmente contendo informações de copyright, links úteis, contato, etc. Assim como o cabeçalho, é um elemento comum a todas as páginas, então vamos criar um componente reutilizável para o rodapé usando o Bootstrap.

Criando o arquivo footer.html:

Dentro da pasta templates, crie um novo arquivo chamado footer.html.

Conteúdo do footer.html:

<footer class="bg-light text-center py-3 mt-5">
    <div class="container">
        <p>© 2024 EmpreGO! - Todos os direitos reservados. </p>
        <p><a href="/contato">Contato</a> | <a href="/sobre">Sobre</a></p>
    </div>
</footer>

Explicação do código:

  • Usamos as classes do Bootstrap para estilizar o rodapé:
    • bg-light: Define a cor de fundo como clara.
    • text-center: Centraliza o texto.
    • py-3: Adiciona um preenchimento vertical de 3 unidades (espaço interno).
    • mt-5: Adiciona uma margem superior de 5 unidades (espaço externo - separando do conteúdo acima).
  • Adicionamos um parágrafo com informações de copyright.
  • Criamos links para as páginas "Contato" e "Sobre" (você pode adaptá-los para as seções relevantes do seu projeto).

Integrando o rodapé no base.html:

Agora, vamos incluir o rodapé no nosso template base (templates/base.html):

<!DOCTYPE html>
<html>
  <head>
    <title>{% block title %}{% endblock %}</title>
    <!-- Link do Bootstrap -->
    <link rel="stylesheet" href="../static/bootstrap/css/bootstrap.css">
    <!-- Link do CSS próprio -->
    <link rel="stylesheet" href="../static/style.css">
    
  </head>
  <body>
    <!-- Cabeçalho -->
    {% include "header.html" %} 

    <!-- Corpo -->
    <main class="container mt-5">
        {% block content %}
        {% endblock %}
    </main>

    <!-- Rodapé -->
    {% include "footer.html" %} 

    <!-- Scripts do Bootstrap -->
    <script src="../static/bootstrap/js/bootstrap.js"></script>
  </body>
</html>

Usamos {% include "footer.html" %} para incluir o código do footer.html no template base, logo após a tag </main>. Agora, todas as páginas que herdarem do base.html terão o mesmo rodapé.

Para você, jovem iniciante: Criar componentes reutilizáveis como o rodapé torna o desenvolvimento mais eficiente e organizado. Você pode personalizar o rodapé com as informações que desejar, como links para redes sociais, endereço, telefone, etc.

Conexão com o projeto EmpreGO!: Com este componente de rodapé, todas as páginas do nosso site terão as mesmas informações de contato e links úteis, garantindo consistência e profissionalismo. No próximo passo, vamos criar o componente de card para exibir as vagas de emprego.
